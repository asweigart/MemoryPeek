# Memory Peek

A single-file, offline-friendly tool for building memory card matching games out of your own images.

Open `memorypeek.html` in any modern browser. Drop in your images, set the board size, pick a backside, and click **Download Game .html** to get a self-contained playable game file with your images baked in.

[Play Memory Peek Online](https://inventwithpython.com/memorypeek/memorypeek.html)

## Features

- Build a memory game from any images you have on disk (PNG, JPG, GIF, WebP, SVG).
- Configurable grid size and copies-per-card (matching pairs, triples, etc.).
- Custom or built-in card backsides.
- Optional HUD: move counter, elapsed timer, matches found.
- Multi-language UI.
- Exports a standalone `.html` file — no server, no dependencies, no internet required to play.

## Philosophy

You own it forever. No accounts, no paywalls, no telemetry, no cloud. Your images stay on your machine. The downloaded game is just a file — share it, archive it, host it anywhere.
